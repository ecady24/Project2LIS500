Project Overview

This project explores how machine learning systems are trained and how bias can emerge through data selection. Using Google's Teachable Machine, we built an image classification model for fruits and analyzed its behavior.

Data Collection

Each group member contributed at 100 original images to the dataset. These images were used to train a classification model.
